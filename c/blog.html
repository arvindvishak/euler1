C is a general purpose programming language introduced in 1972. I grew up on C; my background is in procedural languages. And after not having looked at this language in years, it took me less than five minutes to get back in the saddle and bang out this version of Euler1. Verbose but elegant: 
// Euler1 in C
#include<stdio.h>

int euler(int n) {
    int retval = 0;
    for (int i=0; i<n; i++) {
        if ((i % 3 == 0) || (i % 5 == 0)) {
            retval += i;
        }
    }
    return retval;
}

main() {
    printf("%i\n", euler(1000));
}
My belated thanks to Dennis Ritchie for giving us such a well-designed language. To run this, I used the venerable GCC compiler. Compile to object code, enable execute permissions on the resultant object file, then execute: 
$ g++ euler -o euler1
$ chmod 777 euler1
$ ./euler1
